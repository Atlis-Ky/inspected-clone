Full Stack webstore using react router with functional cart and checkout pages.

- React based architechture, uses router for multipage rendering when needed.
- Products database run on a Postgres backend via Supabase, allows for super easy and secure updating on products & music Data. 
- Functional API calls to the database on supabase rather than hard coded arrays in the project.
- Fully functional calculations and cart logic for all products, with a checkout page to finish orders. 
- Other misc frontend features such as: currency switching, login, discount codes, popups and modals.
- Deployed via netlify
